# Roadtrip2026 Website Brief

## Approved Defaults

- **Stack preference:** Astro
- **Architecture:** multi-page, static-first site
- **Tone:** travel brochure + practical trip planner hybrid
- **Vehicle assumption:** 2025 Tesla Model Y, conservative charging assumptions
- **Charging preference:** Tesla Superchargers preferred
- **Scheduling:** create practical estimates when itinerary timing is not explicit
- **Map approach:** Google-Maps-style route experience, plus a Google Maps route link if possible
- **Version 1 pages:** Home / Overview / Daily Plans / Charging / Map

## Project Goal

Build a Route 66 road-trip website for a 5-day drive beginning May 3, 2026. The site should feel exciting and polished like a travel brochure, but still be useful as a real itinerary/planning tool.

## Required Inputs

Use:
- `GOAL.md`
- `docs/itinerary.md`
- local images in `route66-itinerary/`

## Expected V1 Deliverables

1. A working site scaffold in the repo
2. Shared layout/navigation
3. An itinerary-driven home page
4. Daily pages or a daily detail experience for all 5 days
5. A charging-plan page with realistic structure, assumptions, and clearly labeled estimates
6. A map page prepared for route/stop display and Google Maps handoff
7. Styling that feels modern, readable, and trip-exciting without becoming chaotic

## Important Constraints

- Keep it accessible and responsive
- Avoid heavy complexity unless it earns its keep
- Prefer reusable content/data structures over duplicated markup
- If exact charging numbers are uncertain, label them as estimates and state assumptions clearly
- Preserve local itinerary assets; do not delete existing source images
